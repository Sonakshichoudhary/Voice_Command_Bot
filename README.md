# Voice Command Bot

## Description
A simple Python-based voice assistant that can:
- Tell the current time
- Open YouTube
- Search Google
- Open Chrome browser
- Play music
- Shutdown system

## How to Use
1. Install the required libraries:
   ```
   pip install speechrecognition pyttsx3 pyaudio
   ```

2. Run the bot:
   ```
   python main.py
   ```

3. Give voice commands like:
   - "What's the time?"
   - "Open YouTube"
   - "Search for Python tutorials"
   - "Play music"
   - "Shutdown"
   - "Exit"

## Requirements
- Python 3.x
- Microphone
